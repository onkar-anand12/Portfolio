Project 1

Project Title

Automatic Light Control.

Project Images

Description

The automatic light control project aims to create a system that can adjust lighting levels based on ambient light conditions. The system utilizes a light sensor (such as an LDR) to detect light intensity, a microcontroller (such as Arduino or Raspberry Pi) for processing, and a relay module to control the lights. The circuit is programmed to read the analog input from the light sensor, make decisions based on predefined thresholds, and control the lights accordingly. Testing, calibration, and optional features like time-based scheduling or wireless communication can enhance the functionality of the system. Following safety guidelines and proper documentation is essential throughout the project.

Project 2

Project Title

Self Parking Car

Project Images

Description

The self-parking car project aims to develop a system that enables a vehicle to autonomously maneuver into a parking space without human intervention. The system typically involves various components, including ultrasonic sensors or cameras for detecting obstacles and determining the parking space dimensions, a microcontroller or onboard computer for processing sensor data and controlling vehicle movements, and actuators such as motors for steering, acceleration, and braking. The project begins with designing and implementing the hardware setup, which includes installing sensors at strategic locations on the vehicle and interfacing them with the microcontroller. Next, software development involves programming algorithms to process sensor data, analyze the environment, and generate appropriate steering and motion commands to navigate the vehicle into the parking space safely.

Project Title

Data Driven Irrigation System

Project Images

Description

A conventional irrigation system involves manual or semi-automatic water application to crops using methods like flooding, furrow irrigation, or sprinkler systems. It typically lacks real-time monitoring or precise control and may not optimize water usage efficiently. Drawbacks of conventional irrigation systems include inefficient water usage, soil erosion, runoff, leaching, energy consumption, and labour-intensive operation. The proposed system introduces an intelligent monitoring platform framework based on the Internet of Things, facilitating the transition from traditional to modern farming practices. The proposed system utilizes temperature, humidity, and other sensors to enable smart watering, providing farmers and gardeners with valuable data for soil nurturing. The Internet of Things offers inexpensive, easy-to-install sensors that enhance productivity and efficiency in agricultural practices.
